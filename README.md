# FuraiEngine
C++ゲームエンジンとエディタのプロジェクトです。

## プロジェクト情報
|項目|情報|備考|
|:---|:--|:---|
|OS|Windows10以上||
|開発言語|C++17||
|コンパイラ|Clang++||
|環境|VSCode|C++拡張が必要です。|

## プロジェクトを展開する
現在のプロジェクトは以下の工程が必要です。  
* "Editor/Build/Windows" に"LLVM"(現在の指定バージョンは15.0.0)を配置する。
```text
Editor
 └ Build
    └ Windows
       └ LLVM
```
